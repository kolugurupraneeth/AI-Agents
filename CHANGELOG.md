## Guidelines for Changelog Updates

When making changes, please update this changelog following these guidelines:

- Add changes under the `[Unreleased]` section
- When releasing a new version:
  - Create a new section with the version number and date
  - Move all items from `[Unreleased]` to the new version section
  - Update version numbers in relevant files

### Categories

- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Now removed features
- **Fixed**: Bug fixes
- **Security**: Security fixes and vulnerability patches
